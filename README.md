🦬DataBison-

DataBison is a privacy-first, zero-trust file auditing platform that helps users detect sensitive information, exposed credentials, and hidden metadata before files are shared. DataBison performs all analysis locally in the browser, ensuring zero data uploads and complete user privacy.
